# Price Slider Project

A simple price range slider built with HTML, CSS, and JavaScript.

## Features

- Dual-range slider for selecting minimum and maximum prices
- Real-time price display
- Responsive design
- Prevents invalid ranges (min > max)

## How to Run

1. Open `index.html` in your web browser.
2. Use the sliders to adjust the minimum and maximum prices.
3. The selected price range will be displayed above the sliders.

## Customization

- Adjust the `min`, `max`, and `step` attributes in `index.html` to change the price range and increments.
- Modify `styles.css` to customize the appearance.
- Extend `script.js` to add more functionality, such as filtering products based on the selected price range.

## Browser Support

Works in modern browsers that support HTML5 input range elements.
